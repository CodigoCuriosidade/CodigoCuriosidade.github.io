Programação Desafios

Exemplo de loop usando javascript

Como usar um loop em JavaScript olhando de outro ponto de vista? Um ponto de vista mais interessante podemos dizer… Normalmente aprende-se loops em logica de programação usando exemplos com contas matemáticas, mas curiosos de verdade não são normais, então criei uma solução simples usando loop para mostrar imagens com um efeito simples.

Conhecimentos usados: HTML, CSS, JavaScript e logica de programação.

Código:

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    <style type="text/css">
        
        .classe
        {
            /*Todos os elementos com o class = "classe"
            terao estas caracteristicas*/
            border-radius: 100%; /*Formato de circunferencia*/
            visibility: hidden; /*Esconde os elementos*/
            width: 100px; /*Largura de 100px (pixels)*/
            height: 100px; /*Altura de 100px (pixels)*/
            opacity: 0.10; /*Transparencia*/
           
        }
  
        html
        {
            background-image: url('harry.jpg');   /*imagem de fundo */
            background-size: cover;               /* Essa propriedade faz a imagem preencher todo o espaco do navegador */
        }

        /*Apos o looping terminar os elementos terao estas caracteristicas*/
        .legal
        {
            transition: 10s;
            /* background:rgba(0, 0, 0, 0.8); */
            width: 300px;
            height: 300px;
            float: left;
            padding: 25px;
            margin: 10px;
            border-radius: 100%;
            visibility: visible;
            opacity: 1;

        }
      
    /*Caracteristicas do botao*/
  #btn
  {
    border-radius: 9px;
    font-family: 'Oswald', sans-serif;
    color: #036a81;
    font-size: 135%;
    text-decoration: none;
    padding: 10px 20px;
    border: solid #036881 3px;
    text-transform: uppercase;
    margin: auto;
    position: absolute;
    clear: right;
  }
  #btn:hover
  {
    background-color:#fff;
    border:#fff solid 3px;  
  }
        
    </style>

    <!-- *Elementos* -->
    <img class="classe" src="img/dobby.jpeg" alt="">
    <img class="classe" src="img/harry.jpg" alt="">
    <img class="classe" src="img/albus.jpg" alt="">
    <img class="classe" src="img/hermione.jpg" alt="">
    <img class="classe" src="img/ron.jpg" alt="">
    <img class="classe" src="img/hagrid.jpg" alt="">

    <a href="#" id="btn" onclick="trocar()">Mostrar</a>

    <!-- Script -->
    <script type="text/javascript">
    
        function trocar()
        {
            //Selecionar todos os elementos pelo seletor CSS ".classe"
            //Serao retornados mais de 1 elemento (6 para ser mais //preciso) ja que temos 6 imagens com o class = "classe"
            //Os elementos serao guardados na variavel como se fossem 
            //armazenados em um array e podem ser acessados por index //entre [] ou ()
            var el = document.querySelectorAll('.classe');

            //Selecionar o botao por ID
            var button = document.getElementById('btn');

            //Ao ser clicado o botao ira sumir dentro de 1s (um segundo)
            button.style="visibility :hidden; transition: 1s; ";

            //Loop
            //enquanto o valor de i for menor que o tamanho da variavel //el, cada elemento recebera "legal" como class, ou seja
            //cada elemento tera um class = "legal" substituindo o class //="classe" 

            for (var i = 0; i < el.length; i ++)
            {
                el[i].className = "legal";
            }
        }
    </script>
</body>
</html>
