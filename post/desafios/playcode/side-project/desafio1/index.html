Desafio Playcode: 1-Montando o layout do site

2018.01.25
Desafios

Esses desafios foram feitos para alcançar como resultado um Side Project. Um Side Project é basicamente um projeto de sistemas por exemplo, que você não é obrigado a fazer, você simplesmente faz por dois motivos: 1- Aproveitar o processo de aprendizagem. 2- Ter mais um projeto no GitHub.

Mas veja bem… não é um simples projeto, sim um projeto útil, com coisas que as pessoas poderiam usar para resolver algum problema. Este Side Project foi criado pelo Harlley Oliveira e teria como objetivo mostrar quantos vídeos já foram postados, quais eles são e quantos faltam para completar a sua meta de 100 vídeos do canal Playcode.

Provavelmente a essa altura ele já postou os 100 vídeos, mas da para aproveitar o processo de aprendizagem dentro desse projeto :p



1: Fazer o layout do site

Resolução

Conhecimentos usados para resolver: HTML, CSS e Javascript 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Tarefa 1</title>
</head>
<body>
    <style type="text/css">
    html
    {
        box-sizing: border-box;
    }
    #contador
    {
        float: right;
        margin-right: 80px;
    }
    .post
    {
        float: left;
        width: 265px;
        height: 180px;
        margin: 10px;
        padding: 5px;
    }

    </style>

    <!-- A tag span esta vazia e sera preenchida pelo Javascript -->
    <header>
    <h1>#desafio100videos <span id="contador"></span></h1>
    </header>
    
    <script>
        //Looping para escrever aos 13 videos que faltam 
        //(provavelmente a organizacao dos videos irao ser
        //automatizados mais para frente)
        //f -> faltam
        //p -> postados
        for (var f =0; f<13;f++)
        {
            //escrevendo as 13 imagens
            document.write('<img class="post" src="../img/redirect.png" alt="Video nao postado">');
        }

        //Looping para escrever as 87 imagens que faltam para completar 100 
        for (var p =0; p <87;p++)
        {
            //escrever as 87 imagens
            document.write('<img class ="post" src="../img/img.jpeg" alt="Video-Postado">');
        }

        //Contador para mostrar quantos videos faltam para completar 100
        c = 100 - p;

        //Escrevendo c no HTML
        var mostrar = document.getElementById('contador');
        mostrar.textContent = 'Faltam ' + c + ' videos';
    </script>
</body>
</html>
