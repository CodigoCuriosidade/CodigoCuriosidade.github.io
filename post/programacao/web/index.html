Livros para aprender as bases do desenvolvimento WEB

2017.12.28
Programação
Recomendação

Livros para aprender HTML, CSS e JavaScript

Decidir qual assunto aprender é um grande desafio principalmente para o pessoal da TI, já que a variedade de assuntos são gigantes, mas depois de decidido o que estudar falta agora ir atrás dos materiais necessários para desenvolver essa nova habilidade, mas, quais usar? Foi pensando nessa pergunta que resolvi dizer qual os materiais usei para aprender a desenvolver WEB SITES, hoje apresentarei à você dois livros que li e que estou lendo sobre esse assunto.

Os livros feitos por autoria de Jon Duckett não são capa dura e as folhas são parecidas como capa de revista (Couche ) mas são foscas (TODAS AS FOLHAS *). Ambos podem ser encontrados em português, mas a versão que li/estou lendo é/foi em inglês. O livro de HTML e CSS foi publicado em 2011 e a versão em português foi publicada em 2014. O livro de Javascript e JQuery foi publicado em 2014 e a versão em português foi publicada em 2016.

Começando pelo livro de HTML e CSS o autor conduz o leitor ao processo de aprendizagem de modo gradativo, começando das coisas mais comuns de serem usadas e que são triviais como por exemplo parágrafos, cabeçalhos, colocar vídeos, fotos, áudios ate o Host e tipos de Hosts (Hospedagem e tipos de hospedagens para o site), ou seja, da estrutura de uma pagina ate as perguntas que poderiam te ajudar na entrevista com algum cliente (caso você queira trabalhar de modo freelancer, ou fazer um site pra sua tia ou tio que tem um comercio). Então nesse processo de aprendizagem o leitor aprende a estruturar a pagina e planejar o layout.

No livro de Javascript e JQuery o autor ensina programação básica, por exemplo como fazer operações matemáticas, loopings, operadores lógicos, métodos, funções e orientação a objetos. Já no começo o autor apresenta maneiras de alterar a estrutura HTML a partir da linguagem script, justamente para incentivar a leitura, e com o decorrer do livro os mesmos códigos são retomados de uma forma mais detalhada dando sequencia às maneiras de automatizar processos envolvendo o que já foi aprendido no livro anterior (HTML e CSS). Quando se fala em “Automatizar processos” quer dizer que você irá programar um script para fazer por você o que em tese, seria feito com muito esforço. Por exemplo colocar 100 fotos numa pagina.

Os pontos negativos de ambas as obras é que são caras, logo quem não tem condições terá que “dar um jeito” (procurar em PDF) e não tem exercícios proposto, o que obriga o leitor a buscá-los em outras fontes. Os pontos positivos de ambas as obras são: todas as paginas contem exemplos do assunto que esta sendo tratado naquele momento, neles vem um link que leva ate o site do autor que disponibiliza todos os exemplos feitos tanto para baixar quanto apenas para ver.

Minhas considerações são, os livros são ótimos, o método usado pelo autor foi bem colocado e a maneira detalhada e ao mesmo tempo objetiva chama bastante atenção. O livro não fica enrolando nos assuntos, esses são bem explicados e não deixam nada a desejar quando passamos para o próximo assunto.

Recomendo tanto para quem ja tem certo conhecimento quanto para quem é totalmente leigo, já que os dois livros tem uma didática gradativa, o que possibilita tanto quem não sabe nada quanto quem já sabe alguma coisa, ler com facilidade.
