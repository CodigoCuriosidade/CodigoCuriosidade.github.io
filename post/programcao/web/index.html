Livros para aprender HTML, CSS e JavaScript

Neste primeiro post gostaria de recomendar dois livros que li e recomendo para quem quer aprender HTML/CSS e JS. A velocidade de leitura varia de pessoa para pessoa, mas se você for acostumado com a leitura a segunda parte é ser acostumado a carregar o note para vários lugares (CUIDADO- Grandes riscos de você ser assaltado se morar no Brasil. Se persistirem os sintomas, o médico deverá ser consultado ) A quantidade de livros são incontáveis, o numero de cursos são gigantes, mas para quem gosta de ter um livro (digital ou físico) um ou dois não fazem mal não é verdade?! Passei um bom tempo procurando e pesquisando livros e apostilas sobre HTML e CSS. Claro, isso tomou parte do meu tempo justamente pelo fato de ler e julgar se o método aplicado pelo autor funcionava para mim. No final das contas, encontrei dois livros do mesmo autor, cuja didática é bem diferente de muitos outros livros que li (ou pelo menos comecei). Sendo objetivos mas ao mesmo tempo sendo detalhistas, o autor aborda exemplos sem fotos obrigando o leitor a escrever e “descobrir” como a coisa funciona na prática. Os livros são:

  • HTML e CSS
  • JavaScript&JQuery
Jon Duckett

O preço é um pouco salgado, mas pra quem quer em PDF vale dar uma pesquisada por ai, quem sabe…
